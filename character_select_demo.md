# 🎮 Character Select Screen - Demo Guide

## **Your Threat Digest Now Has a Game-Like Actor Database!**

### **🚀 How to Access**

1. **Generate a digest**: `python launch.py digest --days 7`
2. **Open the HTML file**
3. **Look for the navigation dots** on the left side of the screen
4. **Click the second dot** (Threat Actors) or press `Ctrl+2`

### **🎯 Character Select Features**

#### **Game-Like Interface**
- **Actor cards** with avatars, stats, and preview info
- **Hover effects** with smooth animations and shimmer
- **Color-coded badges** for actor types (Nation State, Financially Motivated, etc.)
- **Stats display** showing techniques, tools, and sectors

#### **Interactive Elements**
- **Click any actor card** → Opens detailed blade with full intelligence
- **Add new actors** → Click the "+" card to search and add new threat actors
- **Navigation dots** → Switch between Digest and Actors pages
- **Keyboard shortcuts** → `Ctrl+1` for Digest, `Ctrl+2` for Actors

#### **Actor Card Information**
- **Avatar** with first letter of actor name
- **Type badge** (Nation State, Financially Motivated, etc.)
- **Quick stats** (number of techniques, tools, sectors)
- **Description preview** (first 3 lines)
- **Alias tags** showing alternate names
